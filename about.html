제품 소개

이동식 랙의 작동 원리부터 기술 유형까지, 모든 것을 알려드립니다

작동 원리

전동 모터로 랙을 좌우로 이동시켜 필요한 곳에만 통로를 생성

1

컨트롤러 조작

전동 컨트롤러 또는 스마트폰 앱으로 원하는 랙 간 통로를 선택합니다. 통로가 열리는 동안 안전 센서가 자동 작동합니다.

2

랙 자동 이동

전동 모터가 레일(또는 바닥 플레이트) 위를 따라 랙을 부드럽게 이동시킵니다. 소음이 거의 없으며 최대 하중 수십 톤까지 지지합니다.

3

지게차 진입 & 작업

통로가 완전히 열리면 지게차가 진입하여 화물을 싣고 내립니다. 작업 완료 후 통로를 닫으면 다시 최대 밀도로 보관됩니다.

Technology Types

현장 조건에 따라 최적의 방식을 선택하세요

RAIL TYPE

레일 방식

바닥에 레일을 매립하여 랙을 이동시키는 방식입니다. 주행 직진성이 높아 대형 창고·냉동창고에 최적이며, 높은 적재 하중을 안정적으로 지지합니다.

  • ✓ 최대 하중 처리에 유리
  • ✓ 냉동·냉장 창고 특화
  • ✓ 직진 이동 정밀도 높음
NO-RAIL TYPE

노레일 방식

바닥 공사 없이 기존 창고에 바로 설치 가능한 방식입니다. 설치 기간이 짧고 이전·재배치가 자유로워 임차 창고나 중·소형 창고에 인기입니다.

  • ✓ 바닥 공사 불필요
  • ✓ 설치·이전 비용 절감
  • ✓ 임차 창고에 적합

Safety First

안전사고 0건을 위한
랙코리아의 3중 안전 시스템

작업자·장비·화물 모두를 지키는 다층 안전 설계

🚨
SENSOR

통로 작업자 감지 센서

적외선·초음파 이중 센서가 통로 내 작업자를 감지하면 랙 이동을 즉시 자동 차단합니다.

🛡️
BUMPER

지게차 충돌 방지 범퍼

랙 하단 전면에 탄성 충격 흡수 범퍼를 장착하여 지게차 접촉 시 구조 손상을 방지합니다.

⛔
STOP

비상 정지 시스템

각 랙 모서리에 비상 정지 버튼 탑재. 누르는 즉시 전체 시스템이 0.3초 내 정지합니다.

🔒
LOCK

낙하 방지 잠금장치

지진·충격 발생 시 자동 잠금되는 낙하 방지 핀으로 화물과 작업자를 동시에 보호합니다.
